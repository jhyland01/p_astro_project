This is a work in progress, the goal is to have a sort of GUI for running the analysis.

  • You should be able to enter values for the number of hyperparameters and number of samples from each hyperparameter, then you should be able to click a button to run the analysis.
  • It would also be nice to have a percentage completion and an output saying at minimum 'success'.
  • There could be some sort of access to the SQL database of samples.
  • There could also be some nice Plotly plots or a produced link to a Tableau cloud.
Assuming we have a situation where someone wants to run this analysis but they don't know python or the command line, then we would want to:
  1. Assume they can download the git repo like you would download anything.
  2. Create an exectuable file that on clicking will create a venv from requirements.txt, the install file.
  3. Create an executable file that will just run python3 -m flask run.
